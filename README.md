# 📊 종목 분석 & 발굴 리포트 생성기 v3

> 종목명 또는 테마를 입력하면 블룸버그 터미널 스타일의 단일 HTML 분석 리포트를 자동 생성합니다. 한국·미국 주식, ETF, 암호화폐를 모두 지원합니다.

---

## 기능 요약

| 입력 예시 | 모드 | 출력 |
|------|------|------|
| `NVDA` / `삼성전자` / `005930` | **A. 개별 종목 분석** | 9섹션 다크모드 대시보드 |
| `소부장 추천` / `AI 반도체` / `K-방산` | **B. 테마 발굴** | 종목 매트릭스 + Top3 픽 |
| `NVDA vs AMD` | **C. 비교 분석** | 좌우 2컬럼 비교 |
| `BTC-USD` | **A 변형 (암호화폐)** | 온체인 지표 기반 |
| `QQQ` | **A 변형 (ETF)** | AUM/보수율 기반 |

핵심 특징
- 단일 HTML 파일 산출물 (Chart.js + Google Fonts CDN만 외부 로드)
- 모든 리포트 우상단에 **📥 PDF 저장 버튼** 자동 포함
- 실시간 웹 검색으로 최신 데이터 확보, 추정치는 `(est.)` 명시
- Chart.js 기반 캔들차트 + EMA 4종 + RSI/MACD 보조지표
- 블룸버그 터미널 스타일 다크 UI (IBM Plex Mono + Noto Sans KR)

## 모드별 리포트 구성

### 🅰️ 모드 A — 개별 종목 9섹션
1. **HEADER** — 티커 배지, 현재가, 8개 메타 (시총·52주 고저·P/E·P/S·배당·실적·컨센서스)
2. **차트 분석 영역** — 매물대 바 + 메인 캔들차트(EMA 4종 오버레이) + RSI/MACD 지표 패널
3. **보조지표 차트** — RSI·MACD 라인/히스토그램
4. **재무 지표 카드** — 매출·순이익·영업이익률·부채비율·현금자산 + 스파크라인
5. **매매 전략 테이블** — 1·2·3차 매수가, 현금 보유, 목표가, 손절가
6. **시나리오 분석** — 강세/중립/하락 3시나리오 (확률 합계 100%)
7. **산업 분석** — 동종업계 비교 + 모멘텀/리스크 태그
8. **핵심 요약 + 결론** — 5개 투자 포인트 + R1~R3 리스크 + 최종 verdict
9. **FOOTER** — 면책 조항 + 검색 출처

### 🅱️ 모드 B — 테마 발굴 7섹션
- 테마 개요 카드 → 종목 매트릭스 (8~12개) → Top 3 픽 카드 (#1 금/#2 은/#3 동) → 4개 분류 그룹 (대장주/성장주/가치주/배당주) → 매크로 환경 분석 → 진입 전략 → 종합 결론

### 🆎 모드 C — 비교 분석
- 모드 A 구조를 좌우 2컬럼으로 분할 + 우월성 매트릭스 (매출 성장 / 이익률 / 부채 / 모멘텀 / 밸류에이션 / 배당 / 미래 전망) + 비중 추천

## 사용 예시

```
# 모르는 분야 → 테마 발굴 → 개별 분석
입력: 소부장 추천
→ 한국 소부장 Top 9 종목 매트릭스 + Top3 픽 카드

입력: 한미반도체
→ 한미반도체 9섹션 상세 분석

# 비교 분석
입력: 삼성전자 vs SK하이닉스
→ 좌우 2컬럼 비교 + 우월성 매트릭스 + 비중 추천

# 옵션 키워드
입력: TSLA 단기      → 차트·이동평균 가중치 ↑, 시나리오 1~3개월
입력: NVDA 심층      → 5개 시나리오로 확장
입력: 소부장 공격적  → 중소형 성장주 위주
```

## 옵션 키워드

티커/테마 뒤에 붙여서 분석 가중치를 조정할 수 있습니다:

| 키워드 | 효과 |
|--------|------|
| `단기` / `트레이딩` | 모멘텀 가중치 ↑, 시나리오 1~3개월 |
| `중기` | 6~12개월 관점 |
| `장기` / `장투` | 펀더멘털 가중치 ↑, 시나리오 3~5년 |
| `심층` | 시나리오를 5개로 확장 |
| `공격적` (모드 B) | 중소형 성장주 비중 70% |
| `안정적` (모드 B) | 대형주 + 배당주 80% |

## PDF 저장

모든 리포트 우상단의 **📥 PDF 저장** 버튼을 누르면:
1. 브라우저 인쇄 다이얼로그가 열림
2. "대상" → **PDF로 저장** 선택
3. "추가 설정" → **배경 그래픽** 체크 (권장)
4. 자동으로 화이트 모드 변환 + 카드별 페이지 분할 방지

차트가 잘릴 경우 용지 방향을 가로(Landscape) 또는 A3로 변경하세요.

## 면책 조항

본 프로젝트가 생성하는 모든 리포트는 **투자 참고용**이며, 투자 손익에 대한 책임은 투자자 본인에게 있습니다. 실제 투자 결정 시 추가적인 실사와 전문가 상담을 권장합니다.
